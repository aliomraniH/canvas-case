# Toolbox review reports

Output directory for the [`toolbox-review`](../skills/toolbox-review/SKILL.md)
skill — one `<YYYY-MM-DD>_review.md` per audit (30-day cadence, or on demand
via `/toolbox-review`). Each report records what was checked, what changed
since the last review, what was flagged, and proposed adds/removals across
skills, debug tools, tests, docs, and setup.

First review is due by **2026-07-12** (TOOLBOX.md was first indexed and
verified 2026-06-12).
